# Homer

## Descripción
Dashboard estático para enlaces de servicios.

## Volúmenes
- `${DATA_DIR}/homer/data` → `/www/assets`

## Variables
Archivo: `.env` raíz (solo `TZ`)

## Endpoint
- https://homer.${BASE_DOMAIN}

## Configuración
`services/homer/config.yml` es estático y ya contiene `winter.errante.es`.
