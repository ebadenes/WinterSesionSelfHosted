# Immich

## Descripción
Gestor de fotos y videos con stack completo (server, microservices, ML, Postgres, Redis).

## Volúmenes
- `${DATA_DIR}/immich/data` → `/data`
- `${DATA_DIR}/immich/data/db` → Postgres
- `${DATA_DIR}/immich/data/redis` → Redis

## Variables
Archivo: `.env` raíz

## Endpoint
- https://immich.${BASE_DOMAIN}

## Fuente oficial
- https://immich.app/docs/install/docker-compose
