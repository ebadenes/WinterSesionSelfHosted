# Pingvin Share

## Descripción
Plataforma de transferencia de archivos.

## Volúmenes
- `${DATA_DIR}/pingvin-share/data` → `/opt/app/backend/data`
- `${DATA_DIR}/pingvin-share/data/images` → `/opt/app/frontend/public/img`

## Variables
Archivo: `.env` raíz (solo `TZ`)

## Endpoint
- https://pingvin.${BASE_DOMAIN}

## Fuente oficial
- https://stonith404.github.io/pingvin-share/setup/installation/
