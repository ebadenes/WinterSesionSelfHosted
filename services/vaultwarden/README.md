# Vaultwarden

## Descripción
Servidor compatible con Bitwarden.

## Volúmenes
- `${DATA_DIR}/vaultwarden/data` → `/data`

## Variables
Archivo: `.env` raíz

## Endpoint
- https://vaultwarden.${BASE_DOMAIN}

## Notas
Habilita 2FA en el panel una vez creado el primer usuario.
